Welcome to Strongloop Web Admin GitHub Pages.

This is a web page for Strongloop Web Admin template. This template will be used as a base project / starting point of every Strongloop application. We are still working on it to make it perfect.

How it works

If you want to run / try this template in your local machine, please take a look some steps how to install all prerequisites and dependencies.

Prerequisite

  • Git
    • Please install Git to your machine. Visit this link to see how to install git.
  • Nodejs
    • Download Nodejs and install it to your machine. Visit this link to download Nodejs based on your environment.
  • Bower
    • Install Bower into your machine with run bellow command:
    $ npm install bower
  • Strongloop
    • Run the following command in your terminal: $ npm install -g strongloop
    *If your environment is windows, please visit this link to see step by step how to install Strongloop. Also, add addional command "--unsafe-perm" when install strongloop. ($ npm install -g strongloop --unsafe-perm)

Troubleshooting
If your environment is Windows and found a problem about NPM behind the proxy, please run the following commands:
$ npm config set registry http://registry.npmjs.org/
$ npm config set http-proxy http://username:password@ip:port
$ npm config set https-proxy http://username:password@ip:port
$ npm set strict-ssl false

Run Web Admin in local machine
After all Prerequisite components already installed into your machine, please do below steps until Web Admin application is running in your machine.
  1. Clone the strongloop_web_admin repository to your machine.
    2. git clone https://github.com/Research-New-Technologies/strongloop_web_admin.git.
  2. Change directory into strongloop_web_admin directory.
    3. cd strongloop_web_admin
  3. Install all dependencies
    4. npm install & bower install
  4. Run the application
    5. node .
  5. Visit the Web application
    6. Go to http://localhost:3000 to see web admin application or to http://localhost:3000/explorer to see Strongloop API explorer.
